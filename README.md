# Relation
A simple program that checks if a given relation is reflexive, symmetric or transitive.

## Reflexive 
A relation is reflexive if every (a,a) ∈ R.

## Symmetric
If (a,b) ∈ R, then (b,a) ∈ R.

## Transitive
If (a,b) ∈ R, (b,c) ∈ R, then (a,c) ∈ R.

## Sample input
n: 3

a: 1
b: 2

a: 4
b: 6

a: 8
b: 9
